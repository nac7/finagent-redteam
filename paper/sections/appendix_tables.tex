\subsection{Statistical Power Analysis}

With $n = 48$ scenarios per model and 10,000 bootstrap resamples:

\begin{itemize}
  \item \textbf{Effect size 0.30 (30 pp)}: Power ≈ 99\% at $\alpha = 0.05$
  \item \textbf{Effect size 0.20 (20 pp)}: Power ≈ 95\%
  \item \textbf{Effect size 0.15 (15 pp)}: Power ≈ 90\%
  \item \textbf{Effect size 0.10 (10 pp)}: Power ≈ 70\%
\end{itemize}

All significant differences reported in the paper are at effect size $\geq 0.20$, ensuring high statistical power.

\subsection{Reproducibility Metadata}

All results are tagged with:
\begin{itemize}
  \item \textbf{run\_hash}: SHA256 of scenario IDs | seed, first 16 chars
  \item \textbf{runner\_version}: ``dev'' for benchmark code version
  \item \textbf{date}: ISO 8601 timestamp of evaluation
  \item \textbf{seed}: Random seed for reproducibility (= 0 for all reported results)
\end{itemize}

Example metadata block from \texttt{results/2026-06-07\_generated-p6\_3trials.json}:

\begin{lstlisting}[style=json]

    "trials": 3,
    "run_hash": "f7ea1376dad932ec",

  }
}
\end{lstlisting}

Researchers can reproduce the exact evaluation by matching run\_hash and seed values.
